# Sawass Ray Tracer

Modern C++ Ray Tracing on-going engine with a compile time resolution favorable approach.

## Features

* Geometry (Static & Dynamic)
* Antialiasing
* Diffuse scattering
* Reflections
* Refractions
* Advanced camera model
* Defocus Blur
* Motion Blur
* Procedural scene generation
* Benchmarking
* PNG image support
