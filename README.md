# Sawass Ray Tracer

Modern C++ Ray Tracing (on-going) engine.

## Features

* BVH
* Procedural textures
* Antialiasing
* Diffuse Scattering
* Reflections
* Refractions
* Advanced Camera Model
* Defocus Blur
* Motion Blur
* Procedural Scene Generation
* Benchmarking
* PNG Image Support
